# 🚀 Kotlin Flow Playground

An interactive Android app built to explore and understand **Kotlin Flow** concepts using real-time UI updates with **Jetpack Compose**.

---

## 📱 Features

- 🔄 Demonstrates core Kotlin Flow operators:
  - map
  - filter
  - debounce
  - collect
  - stateIn / shareIn
- ⚡ Real-time UI updates using Jetpack Compose
- 🧪 Hands-on experimentation with flow transformations
- 📊 Clear visualization of data streams and state changes

---

## 🛠️ Tech Stack

- **Kotlin**
- **Kotlin Flow**
- **Jetpack Compose**
- **Coroutines**
- **MVVM Architecture**

---

## 🎯 Purpose

This project is built as a **learning playground** to:
- Understand how Kotlin Flow works internally
- Visualize how operators transform data streams
- Practice reactive programming in Android
